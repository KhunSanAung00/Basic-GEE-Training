# Create Cloud-Free Composite - Sentinel-2 Images

This short example shows how to filter clouds and create a cloud-free composite for one location using Google Earth Engine JavaScript.

## 1. Define a Location

Choose a point and create a small area of interest around it. You can change the longitude and latitude to any location.

```javascript
// Example location: Bangkok, Thailand
var point = ee.Geometry.Point([100.5018, 13.7563]);

// Create a 10 km buffer around the point
var aoi = point.buffer(10000);

Map.centerObject(aoi, 11);
Map.addLayer(aoi, {color: 'red'}, 'Area of Interest');
```

## 2. Create a Cloud Mask Function

Sentinel-2 images include a `QA60` band. This band contains cloud and cirrus cloud information. The function below removes pixels marked as cloud or cirrus.

```javascript
function maskS2Clouds(image) {
  var qa = image.select('QA60');

  // Bits 10 and 11 are clouds and cirrus clouds
  var cloudBitMask = 1 << 10;
  var cirrusBitMask = 1 << 11;

  // Keep pixels where both cloud and cirrus bits are 0
  var mask = qa.bitwiseAnd(cloudBitMask).eq(0)
      .and(qa.bitwiseAnd(cirrusBitMask).eq(0));

  // Apply mask and scale reflectance values
  return image.updateMask(mask).divide(10000);
}
```

## 3. Load and Filter Sentinel-2 Images

Filter the image collection by location, date, and image-level cloud percentage. Then apply the cloud mask function to every image.

```javascript
var s2 = ee.ImageCollection('COPERNICUS/S2_SR_HARMONIZED')
    .filterBounds(aoi)
    .filterDate('2024-01-01', '2024-03-31')
    .filter(ee.Filter.lt('CLOUDY_PIXEL_PERCENTAGE', 30))
    .map(maskS2Clouds);

print('Filtered Sentinel-2 collection:', s2);
```

## 4. Make a Cloud-Free Composite

Use the median value of each pixel across the filtered collection. This reduces remaining clouds, shadows, and noise.

```javascript
var composite = s2.median().clip(aoi);
```

## 5. Display the Composite

Display the composite using Sentinel-2 true color bands:

- `B4` = red
- `B3` = green
- `B2` = blue

```javascript
var trueColorVis = {
  bands: ['B4', 'B3', 'B2'],
  min: 0,
  max: 0.3
};

Map.addLayer(composite, trueColorVis, 'Cloud-free Sentinel-2 composite');
```

## 6. Full Minimum Code

Copy and paste this complete script into the Google Earth Engine Code Editor.

```javascript
// 1. Define location
var point = ee.Geometry.Point([100.5018, 13.7563]); // Bangkok, Thailand
var aoi = point.buffer(10000);

Map.centerObject(aoi, 11);
Map.addLayer(aoi, {color: 'red'}, 'Area of Interest');

// 2. Cloud mask function
function maskS2Clouds(image) {
  var qa = image.select('QA60');

  var cloudBitMask = 1 << 10;
  var cirrusBitMask = 1 << 11;

  var mask = qa.bitwiseAnd(cloudBitMask).eq(0)
      .and(qa.bitwiseAnd(cirrusBitMask).eq(0));

  return image.updateMask(mask).divide(10000);
}

// 3. Load and filter Sentinel-2 images
var s2 = ee.ImageCollection('COPERNICUS/S2_SR_HARMONIZED')
    .filterBounds(aoi)
    .filterDate('2024-01-01', '2024-03-31')
    .filter(ee.Filter.lt('CLOUDY_PIXEL_PERCENTAGE', 30))
    .map(maskS2Clouds);

print('Filtered Sentinel-2 collection:', s2);

// 4. Create cloud-free composite
var composite = s2.median().clip(aoi);

// 5. Display result
var trueColorVis = {
  bands: ['B4', 'B3', 'B2'],
  min: 0,
  max: 0.3
};

Map.addLayer(composite, trueColorVis, 'Cloud-free Sentinel-2 composite');
```

## Notes

- Change the point coordinates to use another location.
- Change `filterDate()` to use another season or year.
- If the result still has clouds, increase the date range or reduce `CLOUDY_PIXEL_PERCENTAGE`.
- `median()` is commonly used because it creates a stable composite from many images.
