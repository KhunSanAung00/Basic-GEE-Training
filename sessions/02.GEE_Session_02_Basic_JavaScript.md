# Part 1. Welcome to Google Earth Engine (10 min)

## What is Google Earth Engine?

Google Earth Engine (GEE) is a cloud-based platform that allows users to analyze satellite imagery without downloading terabytes of data.

Instead of:

1. Downloading satellite images
2. Storing them locally
3. Processing on your computer

You simply write a few lines of code and Google processes the data in the cloud.

### Why is GEE popular?

- Free for research and education
- Massive satellite archive
- No need for supercomputer
- Fast processing
- Interactive visualization
- Easy sharing

# Part 5. Drawing Objects on the Map (10 min)

## Create a Point

```javascript
var point = ee.Geometry.Point([96.1,19.7]);
Map.addLayer(point);
```

## Create a Rectangle

```javascript
var area = ee.Geometry.Rectangle([95,18,97,20]);
Map.addLayer(area);
```

## Add Color

```javascript
Map.addLayer(area,{color:'red'},'Study Area');
```

# Part 6. Load Your First Satellite Image (10 min)

## Load Sentinel-2

```javascript
var image = ee.ImageCollection(
'COPERNICUS/S2_SR_HARMONIZED'
).first();
```

## Display It

```javascript
Map.addLayer(
image,
{
bands:['B4','B3','B2'],
min:0,
max:3000
},
'Sentinel-2'
);
```

## Zoom to Image

```javascript
Map.centerObject(image,8);
```
