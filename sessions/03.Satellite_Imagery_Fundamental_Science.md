# Part 1. How Do Satellites See the Earth? (10 min)

## Fundamental Science

Satellites do not see objects the same way humans do.

Human eyes can only see Visible Light:

- Blue
- Green
- Red

However, satellites can detect additional wavelengths:

- Near Infrared (NIR)
- Shortwave Infrared (SWIR)
- Thermal Infrared

These wavelengths provide information about:

- Vegetation health
- Water content
- Soil moisture
- Burned areas
- Urban surfaces

---

## Reflection Concept

When sunlight hits the Earth:

Some energy is:

- Reflected
- Absorbed
- Transmitted

Different objects reflect light differently.

| Surface | Reflects Most |
|----------|---------------|
| Healthy vegetation | NIR |
| Water | Blue/Green |
| Soil | Red/SWIR |
| Concrete | Visible + SWIR |

This unique pattern is called a:

## Spectral Signature

Every land cover has its own fingerprint.

# Part 2. Meet Sentinel-2 (10 min)

## Why Sentinel-2?

Sentinel-2 is one of the most popular Earth Observation satellites.

Characteristics:

- 10–20 meter resolution
- Revisits every 5 days
- Free access
- Excellent for vegetation monitoring

---

## Important Bands

| Band | Description |
|--------|------------|
| B2 | Blue |
| B3 | Green |
| B4 | Red |
| B8 | Near Infrared |
| B11 | SWIR1 |
| B12 | SWIR2 |

Memory Trick:

RGB Camera

- B4 = Red
- B3 = Green
- B2 = Blue

Plant Health

- B8 = Near Infrared

# Part 3. Loading Satellite Data in GEE (10 min)

## Load Sentinel-2 Collection

```javascript
var s2 = ee.ImageCollection(
'COPERNICUS/S2_SR_HARMONIZED'
);
```

Print the collection:

```javascript
print(s2);
```

---

## Define an Area of Interest

```javascript
var point = ee.Geometry.Point(
[100.5018,13.7563]
);
```

---

## Filter by Location

```javascript
var images = s2.filterBounds(point);
```

---

## Filter by Date

```javascripta
var images = s2
.filterBounds(point)
.filterDate(
'2025-01-01',
'2025-12-31'
);
```

---

## Count Images

```javascript
print(images.size());
```

# Part 5. False Color Composite (10 min)

## Why False Color?

Human eyes cannot see Near Infrared.

False Color allows us to visualize NIR.

---

## Vegetation Composite

```javascript
Map.addLayer(
image,
{
bands:['B8','B4','B3'],
min:0,
max:3000
},
'False Color'
);
```

---

## Interpretation

| Color | Meaning |
|---------|---------|
| Bright Red | Healthy vegetation |
| Dark Red | Sparse vegetation |
| Black | Water |
| Gray | Urban area |
