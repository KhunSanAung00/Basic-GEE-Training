Google Earth Engine for Beginners

Earth Observation and Remote Sensing Using Google Earth Engine (JavaScript)

Course Overview

A beginner-friendly course introducing Earth Observation, Remote Sensing, Satellite Imagery, and Google Earth Engine (GEE).

Duration: ~10 Hours

Learning Outcomes

Participants will learn to:

  • Understand Earth Observation concepts
  • Understand Remote Sensing principles
  • Access satellite imagery in GEE
  • Create True Color and False Color composites
  • Generate cloud-free composites
  • Calculate NDVI, EVI, NDWI, and NBR
  • Create time-series charts
  • Perform land-cover classification
  • Assess classification accuracy

Course Philosophy

Learn the science first. Understand the satellite second. Write the code third.

The goal is to use Earth Observation to solve real-world environmental and resource management problems.
